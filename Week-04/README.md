# 🚜 Agrotech Catalog

## 📌 Descripción

Catálogo interactivo desarrollado con React y TypeScript para gestionar y visualizar maquinaria agrícola.
Permite realizar búsquedas, aplicar filtros y ordenar resultados en tiempo real.

---

## ⚙️ Funcionalidades

* 📋 Visualización de maquinaria disponible
* 🔍 Búsqueda en tiempo real (case-insensitive)
* 🎛️ Filtro por disponibilidad
* 🔽 Ordenamiento por nombre y precio
* 📊 Contador dinámico de resultados
* ⚠️ Estado vacío cuando no hay coincidencias

---

## 🛠️ Tecnologías

* React
* TypeScript
* Vite
* pnpm

---

## 🧠 Conceptos Aplicados

* Renderizado condicional
* Listas con keys únicas
* Filtrado dinámico
* Ordenamiento sin mutar el array original
* Búsqueda en múltiples campos
* Manejo de estado con useState

---

## ▶️ Ejecución

```bash
pnpm install
pnpm dev
```

---

## 📊 Dominio.

Sistema de catálogo para maquinaria agrícola (**Agrotech**).
