# 🌾 Agrotech - Plataforma de Maquinaria Agrícola

## 📌 Dominio

Aplicación básica para la gestión de maquinaria agrícola, agricultores y alquileres.

---

## 🧩 Entidades Modeladas

* **Machine**: Representa la maquinaria disponible
* **Farmer**: Representa los agricultores (clientes)
* **Rental**: Representa el alquiler de maquinaria

---

## ⚙️ Funcionalidades

* Crear maquinaria
* Registrar agricultores
* Gestionar alquiler de maquinaria
* Controlar disponibilidad de equipos

---

## 🛠️ Tecnologías

* TypeScript
* React + Vite

---

## ▶️ Ejecución

```bash
pnpm install
pnpm dev
```

---

## ✅ Notas

El proyecto implementa:

* Interfaces para modelado de entidades
* Types y type literals para control de datos
* Funciones tipadas para operaciones básicas
