# 📊 Agrotech Dashboard - Datos en Tiempo Real

## 📌 Descripción

Dashboard interactivo desarrollado con React y TypeScript para monitorear maquinaria agrícola en tiempo real.

---

## ⚙️ Funcionalidades

* 📋 Listado de maquinaria (fetch inicial con useEffect)
* 📊 Estadísticas generales (múltiples efectos)
* ⏱️ Actualización en tiempo real (polling cada 5s)
* ⚠️ Manejo de estados: loading, error, data
* 🔄 Cancelación de peticiones con AbortController

---

## 🛠️ Tecnologías

* React
* TypeScript
* Vite
* pnpm

---

## 🧠 Conceptos Aplicados

* useEffect (fetch inicial y polling)
* Cleanup de efectos
* AbortController
* Estados múltiples (loading, error, data)
* Tipado estricto con TypeScript

---

## ▶️ Ejecución

```bash
pnpm install
pnpm dev
```

---

## 📊 Dominio

Sistema de monitoreo de maquinaria agrícola (Agrotech).
